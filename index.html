Распознавание речи из аудиофайлов

Поддерживаемые форматы: WAV, M4A, OGG, MP3, FLAC

Настройки Yandex Cloud

Для работы приложения необходим API-ключ Yandex Cloud:

Получить ключ можно в Yandex Cloud Console. Создайте IAM-токен в разделе "Сервисные аккаунты".

Перетащите аудиофайл сюда или нажмите для выбора

Максимальный размер файла: 10 МБ

Использует Yandex SpeechKit для распознавания речи

Файлы обрабатываются локально, не загружаются на сервер
